FooHTML Yet Another Custom Elements Library

What is FooHTML?

FooHTML is a helper library dedicated for Custom Elements, it allows you to implement and use any new html tags with specific tailor made functionality. Your source code becomes shorter, more easier and semantically clear.

Consider examples in your html code:

<random-image theme="food" />
<random-text min-words="3" max-words="15" />

Browsers support

FooHTML is an experimental project, so keep in mind cross-browser compatibility issues.
It should works fine in Chrome, Firefox and Safari.

<lazy-loading>

Here we have elements, which have build-in lazy-loading concept. It means, they are triggered to change their appearance, when element is first displayed within visible area of the page. Optionally, we can set different offset thresholds.

Scroll down slowly, and see, when and why elements are triggered.

Lazy loading, when top edge is displayed (default behavior). Lazy loading, with 50% of the element's height as top offset. Lazy loading, with 400px top offset Lazy loading, when top edge meets 50% of the screen height. Lazy loading, when 100% of the element's height is displayed. Lazy loading, when element is at the middle of the screen height.
