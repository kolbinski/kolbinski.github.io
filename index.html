FooHTML Yet Another Custom Elements Library

What is FooHTML?

FooHTML is a helper library dedicated for Custom Elements, it allows you to implement and use any new html tags with specific tailor made functionality. Your source code becomes shorter, more easier and semantically clear. Consider examples: <random-image theme="food" /> <random-text min-words="3" max-words="15" />

Browsers support

FooHTML is experimental project, so keep in mind cross-browser compatibility issues. It should works fine in Chrome, Firefox and Safari.

Lazy loading

Lets create custom element with lazy loading build-in. Lazy loading will be triggered with parametrized threashold, depending on element's scrollTop and window's scroll-y position.

Lazy loads, when top edge is displayed (default behavior). Lazy loads, when top edge + 80px is displayed Lazy loads, when 50% of the element's height is displayed. Lazy loads, when 100% of the element's height is displayed. Lazy loads, when top edge meets 50% of the screen height. Lazy loads, when element is at the middle of the screen height.

Video Player

Lets demonstrate more advanced example of inheritance, by creating custom video players (based on HTML5 video tag).

We start with player having basic functionality (title of the movie, play/pause and progress bar) but in addition, when a player is started, other already started player pauses.

Now, lets extend this player and add some more advanced functionality:

  • covers for movies
  • tab on the right allowing to switch between movies
  • subtitles displayed during the "Lorem ipsum" movie
  • adverts displayed during playing the "Dolor sit" movie
  • tiles of all related movies displayed after current movie is finished

There is of cource space for improvements and adding new functionalities, like volume control, showing progress time and video duration, closable adverts, etc.
